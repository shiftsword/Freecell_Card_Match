# FreeCell OCR 纸牌识别系统

一个专门用于识别 FreeCell（空当接龙）游戏截图并生成标准布局的工具，支持图形界面操作。

## 项目概述

本项目通过计算机视觉和模板匹配技术，实现对 FreeCell 游戏截图的自动分析，提取每张纸牌的数字和花色信息，并生成标准的 FreeCell 布局格式，方便玩家记录和分享游戏局面。

## 功能特性

- **图形界面**: 提供直观的用户界面，支持文件选择和剪贴板读取
- **自动分割**: 从游戏截图中精确分割出 52 张纸牌
- **数字识别**: 使用模板匹配技术识别纸牌数字（A, 2-9, T, J, Q, K）
- **花色识别**: 基于点数锚定的花色提取 + 模板匹配，识别四种花色（H♥, S♠, D♦, C♣）
- **多分辨率支持**: 自动检测卡片尺寸，选择对应分辨率的模板集
- **发光效果处理**: 内置黄色/金色发光像素过滤，处理游戏界面装饰效果
- **布局生成**: 输出标准的 FreeCell 布局格式
- **完整性验证**: 自动检查牌组是否完整合法（52张，每种花色13张）
- **自动复制**: 识别结果自动复制到剪贴板，方便分享
- **模板创建**: 支持创建和管理多套识别模板，提高适应性

- OpenCV (`opencv-python`)

- Pillow
- Tkinter（GUI 界面）

```bash

```

```bash

```

1. 在图形界面中选择游戏截图文件，或从剪贴板读取

3. 点击"匹配识别"按钮
4. 识别结果自动复制到剪贴板

## 技术实现

### 处理流程

```
游戏截图 → 发光效果过滤 → 卡片分割 → 数字提取 + 花色提取 → 模板匹配 → 布局生成 → 验证
```

1. **发光效果过滤**：检测并抑制黄色/金色发光像素（HSV H:15-35, S:50-255, V:150-255），防止干扰颜色检测
2. **卡片分割** (`card_splitter.py`)：通过 HSV 颜色空间检测绿色背景，定位 8 列纸牌区域，使用中位数宽度统一裁切，消除发光效果导致的尺寸不一致
3. **数字提取** (`extract_numbers.py`)：在卡片左上角 ROI 区域检测红/黑颜色轮廓，提取数字区域并转换为白底黑字格式
4. **花色提取** (`extract_numbers.py`)：定位点数底部，在其下方左侧限定区域（x < 20% 宽度）提取花色符号，排除卡片中央大花色干扰
5. **模板匹配** (`match_numbers.py`)：使用归一化相关系数（TM_CCOEFF_NORMED）匹配点数和花色模板，支持多模板集自动回退
6. **布局生成** (`match_numbers.py`)：将识别结果排列为 8 列布局，验证 52 张牌的完整性

### 核心模块

| 文件 | 功能 |

| `Freecell_Card_Match_GUI.py` | 图形用户界面主程序 |

### 花色识别算法

花色识别是本系统的核心技术难点。卡片结构如下：

```
┌──────────────────────┐
│ 数字  ← 点数区域      │
│ 花色  ← 目标区域      │  搜索 x < 20% 宽度
│                      │
│     大花色 ← 干扰源   │  x > 20% 自动排除
└──────────────────────┘
```

1. **点数锚定**：先定位左上角的点数区域底部
2. **限定搜索**：在点数下方、卡片左侧 20% 宽度内搜索花色符号
3. **分辨率自适应**：所有参数基于卡片尺寸比例计算，支持不同分辨率
4. **模板匹配**：将提取的花色与预建模板匹配，识别 H/S/D/C

```python
scale = height / 58.0  # 相对于标准 58 像素高度的缩放比
rank_y_limit = int(height * 0.26)     # 点数 y 范围上限
rank_x_limit = int(width * 0.17)      # 点数 x 范围上限
search_x_max = int(width * 0.20)      # 花色搜索宽度上限
search_height = int(height * 0.38)    # 花色搜索高度
```

### 发光效果处理

游戏截图中的黄色/金色发光效果会影响识别准确率：

| 图像 | 发光像素占比 | 卡片尺寸分布 | 识别率 |
|------|-------------|-------------|--------|
| Freecell_Layout.png | 0.53% | 148-149×58 | 100% |
| Freecell_Layout_1.png | 0.94% | 223-238×87 | 100% |

处理措施：
1. **颜色检测阶段**：从红色掩码中去除 HSV H:15-35 的发光像素
2. **卡片分割阶段**：使用中位数宽度统一裁切，消除发光边缘干扰
3. **模板匹配阶段**：支持多模板集自动回退，适应不同视觉风格

### 模板集

系统为不同分辨率维护独立的模板集：

| 模板集 | 卡片尺寸 | 适用截屏 |
|--------|---------|---------|
| `Card_Rank_Templates/set_1920x1080` | ~149×58 | 1920×1080 分辨率 |
| `Card_Rank_Templates/set_2880x1800` | ~231×87 | 2880×1800 分辨率 |
| `Card_Suit_Templates/set_1` | ~149×58 | 1920×1080 分辨率 |
| `Card_Suit_Templates/set_2880x1800` | ~231×87 | 2880×1800 分辨率 |

每个花色模板集包含 52 张模板（每种花色 13 张），经 AI 视觉逐张校验标签。
